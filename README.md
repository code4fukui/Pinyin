# Pinyin.js

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A JavaScript library for converting Pinyin with tone numbers to Unicode characters with tone marks.

## Usage

```js
import { Pinyin } from "https://code4fukui.github.io/Pinyin/Pinyin.js";

console.log(Pinyin.decode("ni1")); // Output: "nī"
```

## Features

- Converts Pinyin with tone numbers to Unicode characters with tone marks
- Supports all Mandarin Chinese vowels and tone marks
