# Pinyin.js

声調番号付きのピンインを、声調記号付きのUnicode文字に変換するJavaScriptライブラリです。

## 使い方

```js
import { Pinyin } from "https://code4fukui.github.io/Pinyin/Pinyin.js";

console.log(Pinyin.decode("ni1")); // 出力: "nī"
```

## 特徴

- 声調番号付きのピンインを、声調記号付きのUnicode文字に変換
- 標準中国語のすべての母音と声調記号をサポート
